قائمة المنتجات المقاطعة مشروع مبني على (لارافيل) وأدوات أخرى أهمها (ريكات) ويهدف هذا المشروع لعرض أي مُنتج يمكن تصنيفه ضمن القائمة، كما يُوفر واجهة بحث مرنة وسهلة الاستخدام لنتأكد أن المُستخدم وجد الإجابة التي يبحث عنها في وقت قياسي.

### الميزات الحالية
- واجهة إدارة مبنية على (فيلامنت)
- ميزة البحث عبر الكلمات المفتاحية
- صفحة لكل منتج وكل شركة

### كيف نشغل الموقع في الجهاز؟
- تأكد من تثبيت (composer) ومُدير حِزم مثل (npm)
- قم بتنزيل الحِزم
```
composer install
npm install
```
- غير ما يلزم من الإعدادات في ملف .env وقم بإنشاء مفتاح
```
php artisan key:gen
```
- قم بتشغيل الموقع
```
composer run dev
```
والآن يفترض بالموقع أن يكون مُتاحًا عبر http://localhost:8000

### الميزات التي نرغب بها
- إضافة باركود لكل منتج بحيث يمكن طباعته وإعطاؤه لأي شخص اشترى مُنتجًا من المنتجات المقاطعة
